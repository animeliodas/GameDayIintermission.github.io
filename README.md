# MEGA QIZ — Speedrun Edition 🎮

Игра-угадайка: чей мировой рекорд быстрее?  
Веб-версия Delphi-приложения, готовая для **GitHub Pages**.

## Как играть

На экране — две карточки с играми со speedrun.com.  
На левой карточке видно **реальное** время мирового рекорда.  
На правой — скрыто (`???`).  
**Нажми на ту карточку, чей рекорд, по-твоему, быстрее.**  
Правильно — +1 к счёту, неправильно — −1.

## Структура файлов

```
index.html          ← вся игра (HTML + CSS + JS)
Pre-created.json    ← заранее заготовленные игры (быстрый старт)
README.md
```

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub (например `speedrun-quiz`).
2. Загрузи `index.html`, `Pre-created.json`, `README.md`.
3. Зайди в **Settings → Pages → Source** → выбери ветку `main`, папку `/ (root)`.
4. Сохрани. Через ~1 минуту игра будет доступна по адресу:  
   `https://<твой-юзернейм>.github.io/<репозиторий>/`

## Pre-created.json — формат

```json
{
  "games": [
    {
      "id":       "уникальный slug или id",
      "GameName": "Название игры",
      "CatName":  "Название категории",
      "WRTime":   "PTxHxMxS",
      "Image":    "https://ссылка-на-обложку"
    }
  ]
}
```

`WRTime` принимается в формате ISO 8601 (`PT1H2M3S`) **или** `M:SS` / `H:MM:SS`.

## Как пополнить Pre-created.json

Запусти оригинальное Delphi-приложение — оно сохраняет  
загруженные игры в `Pre-created.json` рядом с `.exe`.  
Скопируй этот файл в репозиторий.

## Технические детали

| Что | Как |
|-----|-----|
| Рандомные игры | `GET /api/v1/games?max=1&offset=<random>` |
| CORS | сначала прямой запрос, при ошибке — через `corsproxy.io` |
| Параллельность | `Promise.all` для обложки + категории |
| Масштабирование | CSS `transform: scale()` — точное повторение Delphi ScaleLayout |
| Очередь | Fisher-Yates shuffle, без повторений подряд |

## Ограничения

- speedrun.com API rate-limit: **100 запросов/минуту**.  
  При превышении игра автоматически делает паузу 12 секунд.  
- Без `Pre-created.json` первая пара появится через ~10–20 секунд  
  (время загрузки двух игр с API).
